# Nav2 파라미터 변경 내역

## 1. controller_server > goal_checker

| 파라미터 | 변경 후 | 변경 전 | 설명 |
|----------|----------|----------|------|
| stateful | false | true | 목표 도달 조건을 지속적으로 유지할지 여부 |
| xy_goal_tolerance | 0.10 | 0.25 | XY 목표 허용 오차 (m) |
| yaw_goal_tolerance | 0.05 | 0.25 | 방향각 목표 허용 오차 (rad) |

### 변경 목적
- `stateful` → 매 순간 새로 판정하여 더욱 정확한 목표 도달 판정 수행
- `xy_goal_tolerance` → 목표 위치에 더 정확히 도착하도록 설정
- `yaw_goal_tolerance` → 목표 방향에 더 정확히 정렬 후 정지하도록 설정

## 2. controller_server > FollowPath

| 파라미터 | 변경 후 | 변경 전 | 설명 |
|----------|----------|----------|------|
| plugin | RegulatedPurePursuitController | dwb_core::DWBLocalPlanner | 경로 추종 알고리즘 변경 |

### 변경 목적
- 이동과 회전을 동시에 수행하는 방식 대신
  **회전 → 직진 이동 → 최종 회전** 순서로 동작하도록 변경
- 경로 추종 안정성 향상
- 좁은 공간에서 불필요한 진동(Oscillation) 감소

---

## 3. local_costmap > inflation_layer

| 파라미터 | 변경 후 | 변경 전 | 설명 |
|----------|----------|----------|------|
| inflation_radius | 0.15 | 0.50 | 장애물 팽창 반경 (m) |
| cost_scaling_factor | 10.0 | 5.0 | 팽창 비용 감소율 |

### 변경 목적
- 좁은 맵 환경에 맞춰 장애물 팽창 영역 축소
- 벽과 지나치게 멀어지지 않고 주행 가능
- 벽면을 따라 안정적으로 이동할 수 있도록 설정

---

## 4. global_costmap > inflation_layer

| 파라미터 | 변경 후 | 변경 전 | 설명 |
|----------|----------|----------|------|
| inflation_radius | 0.25 | 0.50 | 장애물 팽창 반경 (m) |

### 변경 목적
- 전역 경로 생성 시 장애물과 일정 거리를 유지
- 경로 자체는 벽에서 약간 떨어진 위치로 생성되도록 설정
- 안전성과 공간 활용성의 균형 확보

---

## 기대 효과

- 목표 위치 및 방향 정밀도 향상
- 회전과 이동이 분리되어 보다 자연스러운 주행 수행
- 좁은 통로에서 경로 생성 및 추종 성능 향상
- 벽 근처 주행 시 불필요한 회피 동작 감소
- 전체적인 Navigation 안정성 향상
