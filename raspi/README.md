# Raspi

### 🔌 Raspberry Pi 5 ↔ STM32 UART 결선 정보

라즈베리파이와 STM32 간의 유선 핸드셰이크 통신을 위한 핀 맵입니다. 반드시 서로의 **TX와 RX를 교차(Cross)**하여 연결하고, **GND를 공유**해야 합니다.

#### 1. GPIO 직접 연결 시 (소스코드 포트: `/dev/ttyAMA0`)
| 라즈베리파이 5 핀 번호 | 핀 이름 (기능) | 방향 | STM32 타깃 핀 |
| :---: | :--- | :---: | :--- |
| **8번 핀** | GPIO 14 (TXD) | ➡️ | MCU 수신 핀 (RX) |
| **10번 핀** | GPIO 15 (RXD) | ⬅️ | MCU 송신 핀 (TX) |
| **6번 핀** | GND (접지) | ↔️ | MCU GND (접지) |

#### 2. USB to UART 모듈 사용 시 (소스코드 포트: `/dev/ttyUSB0`)
| USB to UART 모듈 핀 | 방향 | STM32 타깃 핀 |
| :--- | :---: | :--- |
| **TX** (송신) | ➡️ | MCU 수신 핀 (RX) |
| **RX** (수신) | ⬅️ | MCU 송신 핀 (TX) |
| **GND** (접지) | ↔️ | MCU GND (접지) |

> ⚠️ **주의사항**
> - 라즈베리파이의 GPIO는 **3.3V 전압 레벨**로 동작합니다. STM32의 5V 전원 핀이 라즈베리파이의 RX 핀으로 잘못 들어가지 않도록 주의하세요.
> - 두 보드 모두 각자 전원을 공급받고 있으므로, **VCC(5V/3.3V) 선은 서로 절대 연결하지 마십시오.** (쇼트 위험)
